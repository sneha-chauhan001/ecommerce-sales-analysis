#  E-Commerce Sales Analysis & Dashboard

##  Overview

This project explores an e-commerce sales dataset to understand how products perform, how customers behave, and how revenue is generated across different regions.

The goal was to perform data analysis using Python and then build a simple, interactive dashboard using Power BI to present the insights in a clear and visual way.



##  Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* Power BI
* Excel (basic understanding and structure)



##  What was analyzed

* Top product categories and most ordered items
* Revenue distribution across different regions
* Customer demographics (age and gender)
* Payment method preferences
* Monthly sales trends
* Return patterns
